PHP-GeoRev-Class by gplv2

PHP server side reverse geocode class

A php5 class to perform reverse serverside geocoding request using the json capabilities of the following engines: - Google - Yahoo - Nominatim - Bing - Geonames

Dependencies

For Memcache support you will need the provided class.memcached.php file

Install

As usual, drop it somewhere in the php path of your application or in the workdirectory and check out the provided sample
